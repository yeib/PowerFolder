## 🚀 ¿Qué es PowerFolder?

PowerFolder es el puente definitivo entre **la velocidad implacable de la terminal** y **la comodidad visual de una interfaz gráfica moderna**. Si alguna vez sentiste que el explorador nativo de Windows es demasiado lento, torpe, o requiere demasiados clics para tareas simples, **PowerFolder es para ti**.

## ✨ Características Principales (Próximamente)

* 🔍 **La Barra Omnipotente**: Presiona `Ctrl + Espacio` para invocar una paleta de comandos estilo *Spotlight/Alfred*. Escribe `zip fotos`, `resize 50%` o `rename` y deja que el explorador lo haga todo sin tocar el mouse.
* ⌨️ **Keyboard-First**: Navegación en cascada y atajos globales. Usa el explorador a toda velocidad sin soltar el teclado.
* 🎨 **Renombrado Masivo Visual**: Reglas avanzadas, soporte de Regex y Live-Preview de cómo quedarán tus archivos antes de confirmar.
* 📦 **Gestor de Medios y Comprimidos Nativo**: Explora `.zip` y `.rar` como si fueran carpetas normales y convierte imágenes (WEBP, PNG, JPG) con un solo clic.

## 🛠️ Tecnología bajo el capó

PowerFolder está diseñado obsesivamente para consumir la mínima RAM posible y abrir en milisegundos.
- **Tauri & Rust**: Para interactuar directamente con el sistema de archivos a niveles bajos con rendimiento nativo extremo.
- **React & TypeScript**: Para ofrecer una interfaz *Glassmorphic* vibrante y fluida (a 60fps).

## 💡 Estado del Proyecto

PowerFolder se encuentra en fase activa de desarrollo **[Closed Beta]**. Este repositorio público sirve como landing page informativa. El núcleo de la aplicación (`powerfolder-core`) se mantiene privado mientras pulimos la magia.
