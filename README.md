## 🚀 ¿Qué es PowerFolder?

PowerFolder es el puente definitivo entre **la velocidad implacable de la terminal Linux/Rust** y **la comodidad visual de una interfaz gráfica moderna**. Si alguna vez sentiste que el explorador nativo de Windows es demasiado lento, torpe, o requiere demasiados clics para tareas simples, **PowerFolder es para ti**.

## ✨ Power Tools (Ya Implementadas)

No es solo un explorador; es un arsenal de productividad. Todas las funciones se ejecutan localmente a máxima velocidad, sin subir tus archivos a la nube:

* 🪟 **Paneles Duales (Split View):** Divide tu pantalla y lanza archivos de una carpeta a otra como un profesional del *drag & drop*.
* 🔍 **Buscador Semántico (Deep Search):** Encuentra una palabra específica *dentro* del texto de cientos de documentos, scripts y logs al instante gracias a procesamiento multi-hilo (`rayon`).
* 🔒 **Bóveda Secreta (Vault):** Cifra y oculta tus archivos más sensibles con encriptación de grado militar AES-256 GCM directamente desde el menú de clic derecho.
* 📝 **Extracción OCR & Compresión de Video:** Convierte el texto de las imágenes en texto seleccionable en segundos, o comprime videos MP4 masivos con el poder nativo de `FFmpeg`, todo a un clic de distancia.
* 🗑️ **Limpieza Extrema (EXIF y CSV):** Destruye el rastreo GPS y la metadata de tus fotos personales para privacidad absoluta, o purga las filas vacías de tus bases de datos gigantes al instante.
* ⏱️ **Productividad Silenciosa:** Un *Time Tracker* pasivo contabiliza cuánto tiempo trabajas en cada carpeta. Además, incluye un *Mouse Jiggler* (`Keep-Alive`) para evitar que tu PC se suspenda o aparezcas como "Ausente".
* 🎨 **Renombrado Masivo Visual:** Cambia el nombre de 500 archivos de golpe.
* 📋 **Historial de Portapapeles:** Presiona `Ctrl+Shift+V` para acceder a todo lo que has copiado y nunca volver a perder una URL o un texto importante.

## 🛠️ Tecnología bajo el capó

PowerFolder está diseñado obsesivamente para consumir la mínima RAM posible y abrir en milisegundos.
- **Tauri & Rust:** Para interactuar directamente con el sistema de archivos a niveles bajos con rendimiento nativo extremo y multi-hilo.
- **React & TypeScript:** Para ofrecer una interfaz moderna, vibrante y fluida.

## 💡 Estado del Proyecto
